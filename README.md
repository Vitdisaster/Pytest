# pet-проект по автоматизации

- Автоматизация тестирования API с помощью python+pytest
- Проверка схем ответов с помощью jsonschema/pydantic


Реализовано:
Подключение к тестовому сайту для работы с REST API
Валидация ответа запроса по json-схеме и с pydantic
Генерация allure отчетов

Используется:
Python 3.14
Pytest 9.0
requests
jsonschema
pydantic
allure
